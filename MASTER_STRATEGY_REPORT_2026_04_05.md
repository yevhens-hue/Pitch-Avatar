## 1. Фундаментальные задачи (P0 — Критично для бизнеса)
Это функции, которые превращают Pitch Avatar из индивидуального сервиса в мощную B2B-платформу.

| Группа | Задача | Описание и UX эффект |
| :--- | :--- | :--- |
| **Integrations** | **Integrations Hub** | Создание страницы `/integrations`. Salesforce, HubSpot, Zapier. Критично для автоматизации продаж (ROI). |
| **Dev Tools** | **API & Webhooks** | Раздел в профиле для генерации ключей и настройки Webhooks (передача лидов в реальном времени). |

| **Data** | **Real Analytics** | Замена моков на интерактивные графики (`recharts`). Анализ вовлеченности по каждому слайду. |

## 2. Оптимизация UX и Конверсии (P1 — Рост показателей)
Улучшения, направленные на сокращение пути пользователя и максимизацию лидов.

### 🚀 Сокращение пути к ценности (Time-to-Value)
*   **One-Click Upload**: Карточки на Dashboard превращаются в активные Dropzone. Загрузка файла начинается мгновенно без лишних модалок.
*   **Мгновенное превью**: После загрузки файла — 5-секундное приветствие аватаром по имени, чтобы показать мощь технологии "здесь и сейчас".
*   **Библиотека шаблонов**: Вместо пустого экрана в "Create from scratch" — готовые сценарии (Sales Pitch, HR, Marketing).

### 📈 Инструменты конверсии (Conversion Boosters)
*   **Verbal CTAs**: Голосовые призывы аватара («Нажмите кнопку справа») синхронизированные с появлением элементов на слайде.
*   **Геймификация лид-форм**: Сбор данных через интерактивный диалог с аватаром, а не через статичную форму.
*   **Smart Links & Personalization**: Динамические приветствия через URL (`?name=Ivan`) и разный контент для LinkedIn/Facebook.
*   **Интеграция Календаря**: Всплывающее окно Calendly прямо в плеере, когда аватар предлагает назначить встречу.

## 3. Оптимизация и упрощение Flow (Удаление лишнего)
Анализ того, что можно убрать или упростить для ускорения работы.

*   **Flow: Слайды**: Скрыть «Расширенные настройки» в модалке по умолчанию. Показывать их только после успешного парсинга файла (Progressive Disclosure).
*   **Flow: Видео**: Объединить дропзоны для Слайдов и Видео. Система должна сама определять тип файла (.pptx vs .mp4), убирая лишний клик по табам.
*   **Flow: С нуля**: Убрать из модалки. Сделать прямым переходом в редактор с дефолтным аватаром.
*   **Flow: AI-ассистент**: Объединить шаги «Инструкции» и «База знаний». Сейчас это воспринимается как слишком длинный путь.
*   **Sidebar Cleanup**: Сгруппировать «Помощь», «Роли», «База знаний» под общий заголовок «Настройки ИИ».

## 4. Техническое совершенство и UI (P2 — Качество продукта)

### 🎨 Интерфейс и Визуал
*   **Тёмная тема (Dark Mode)**: Реализация через CSS-переменные. Маст-хэв для Dev-аккаунтов.
*   **SVG Иконки**: Замена всех эмодзи (🏠, 📂) на библиотеку `lucide-react`.
*   **Микро-анимации**: Плавные переходы (fades/slides) между шагами визардов для ощущения "дорогого" продукта.
*   **Контрастность**: Корректировка синего цвета `#0070f3` до стандарта WCAG AA (4.5:1).

### 🛠️ Код и Архитектура
*   **Universal UI Kit**: Создание общих компонентов `Button`, `Input`, `Table` в `src/components/ui`. Устранение 40% дублей.
*   **Advanced Editor Canvas**: Внедрение движка (Konva/Fabric) для свободного перемещения аватара и слоев элементов.
*   **Zustand State**: Сохранение черновиков на каждом шаге создания, чтобы данные не терялись.
*   **i18n**: Полная локализация для устранения смеси RU/EN в Dev-интерфейсе.
*   **История версий**: Добавление Undo/Redo в редактор презентаций.

## 6. Психология продукта: Aha! Моменты и Активация
Фокус на скорости получения первого «живого» результата и осознании бизнес-ценности.

### ✨ Aha! Моменты (Мгновенная ценность)
*   **Visual Magic**: Первое видео-превью за 30 секунд. Увидеть реалистичного аватара, произносящего твой текст.
*   **Multilingual Power**: «Я говорю на японском!». Мгновенный перевод презентации с клонированием голоса.
*   **Brain & Face**: Первый точный видео-ответ аватара на сложный вопрос из загруженной базы знаний.
*   **Content Alchemy**: Авто-генерация продающего сценария ИИ на основе «сухого» технического PDF-файла.
*   **Seamless Identity**: Момент, когда пользователь слышит *самого себя* (через клонирование голоса), идеально говорящего на иностранном языке.

### ⚡ Активация (Ключевые действия)
*   **Soft Activation**: Публикация первой ссылки и её копирование в буфер обмена.
*   **Engagement Activation**: Первый просмотр презентации внешним зрителем, зафиксированный в Analytics.
*   **Hard Activation (ROI)**: Получение первых контактных данных лида через интерактивную форму в плеере.
*   **Automation Loop**: Успешная интеграция с CRM и получение первого тестового лида в Salesforce/HubSpot.
*   **Viral Loop**: Приглашение первого коллеги в команду (Company Users) и создание совместного проекта.
