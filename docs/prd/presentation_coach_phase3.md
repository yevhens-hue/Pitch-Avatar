# Pitch Avatar: Sales Coaching Platform (Coach UI · Phase 3)

*Основано на презентации (Июнь 2026).*

## Общая концепция
Pitch Avatar расширяется до полноценной **Sales Coaching Platform**. 
Внедряется новый тип проекта в Wizard (Coach Avatar).
Разработка разбита на 3 волны (Waves).

## Волна 1 (Wave 1): Wizard + Editor
**1–2 спринта (Июнь-Июль)**
1. **Coach Avatar toggle на step 3**: Без нового шага — тогглер Да/Нет + выбор роли ученика (Role selector).
2. **Coach Q&A tab в редакторе**: Новая вкладка в правой панели редактора слайда.
3. **AI генерация Q&A**: Изолированный от Knowledge Base генератор вопросов, учитывающий выбранную роль ученика.

## Волна 2 (Wave 2): Listener + Settings
**1–2 спринта (Июль-Август)**
1. **Listener Coach mode**: Режим слушателя. Score pill, отображение правильного ответа, UI для реального клиента.
2. **Settings (Visibility + Reporting)**: Настройки показа фидбека и интеграции (Email/Slack/HubSpot).
3. **Editor без слайдов (Quiz)**: Режим, где Q&A bank выступает как основной canvas, а панель слайдов пустая.

## Волна 3 (Wave 3): Advanced
**1 спринт (Август-Сентябрь)**
1. **Slide-as-answer**: *Ключевая зависимость (нуждается в Stage 2 — media-data triggers).*
2. **4 режима тренировки**: Добавление Mode 3 и Mode 4.
3. **Custom roles библиотека**: Пользователи могут добавлять свои кастомные роли для учеников.

---

## Матрица режимов тренировок (2x2)

Концептуальная матрица: Кто (Coach/Ученик) × Что делает (Рассказывает/Спрашивает).

1. **Coach рассказывает (Пассивное обучение):** Аватар презентует контент по слайдам. Слушатель смотрит и слушает. Оценка знаний — отдельно потом. *(Use case: онбординг новичка, обучение новым продуктам)*.
2. **Coach спрашивает (Тест / Экзамен - DEFAULT):** Аватар играет роль клиента и задает вопросы. Слушатель отвечает. LLM проверяет качество ответа. *(Use case: аттестация продавца перед клиентом)*.
3. **Ученик рассказывает (Pitch practice):** Слушатель презентует продукт (pitch). Аватар в роли клиента слушает и оценивает структуру, тон, полноту. *(Use case: подготовка к первой встрече)*.
4. **Ученик спрашивает (Q&A обучение):** Слушатель задает любые вопросы. Аватар отвечает из Knowledge Base. Быстро освежить знания перед встречей. *(Use case: быстрая справка для продавца)*.

## Детализация UI / Мокапы

### 1. Project List
- Новая колонка **Mode** рядом с Type. У Coach-проектов иконка гантели (🏋). Для обычных — дефис.
- Новый dropdown-фильтр сверху: `Mode: Coach / All`.

### 2. Wizard Step 3 (Avatar Instructions)
- Появляется чекбокс **Coach Mode**.
- При включении отображается выпадающий список **Роль ученика** (например: Account Executive).
- В левом сайдбаре (навигации) сразу после шага 3 появляются 2 новых шага (выделены янтарным цветом): `Coach Q&A Set` и `Coach Settings`. Существующий шаг `Knowledge base` не меняется, просто сдвигается ниже (на позицию 6).

### 3. Wizard Step 4 (Coach Q&A Set - NEW)
- **Цель:** Формирование набора вопросов и ответов независимо от Knowledge base. 
- **Источник контента:** Отдельная копия UI выбора контента (Upload file, Add URL, Text), которая используется только для генерации Q&A с учетом роли ученика. Контент сохраняется в изолированный `coach_qa` scope.
- **Параметры генерации:** Кнопка «Generate & add to Set», настройки: количество вопросов (число), сложность (dropdown), тип вопросов (чипы/мультиселект).
- **Test Set:** Таблица/список сгенерированных и добавленных вручную вопросов с кнопками Edit, Delete, Import CSV, Add manually.

### 4. Wizard Step 5 (Coach Settings - NEW)
- **Тип Test (формат проверки):** `Text/voice`, `Text + правильний слайд`, `Лише правильний слайд`.
- **Выбор Q&A из Set:** Чекбоксы по категориям (Pricing, Objection, Tech) и ограничение количества вопросов.
- **Когда задавать (Timing):** `До`, `На слайдах`, `Після`, `Без слайдів`. Порядок: `Послідовно`, `Рандомно`.
- **Три независимых флага показа (Visibility):** Оценить правильность сразу, Показать правильный ответ после каждого, Показывать текущий балл постоянно.
- **Passing Score:** Пороговый балл в %.
- **Отчетность:** Чекбоксы Trainer, Learner, Slack, HubSpot.

### 5. Editor - Top Nav & Coach Q&A tab
- **Навигация:** В верхнем меню после `Goals` добавляются 2 новых пункта: `Coach Q&A Set` и `Coach Settings` (реюз тех же компонентов, что и в Wizard).
- **Правая панель:** Новая вкладка `Coach Q&A` рядом со `Script` и `Elements`. Показывает вопросы для текущего слайда. Есть стрелки (▲▼) для изменения порядка вопросов и кнопка `+ Add Q&A from Set`.

### 6. Editor Preview - Train Mode (AI asks / Manual)
- В режиме Preview появляется sticky-баннер сверху с чекбоксом **Train Mode**.
- **Sub-toggle [AI asks / Manual]**:
  - **Режим AI asks:** Аватар задает вопрос. Тренер пишет правильный ответ в чат. LLM автоматически категоризирует и определяет сложность. Выводится зеленый бейдж: `✓ SAVED · Q6 записано у Test Set`.
  - **Режим Manual:** Вместо чата выводится форма ручного ввода (Textarea для вопроса, Textarea для ответа, dropdowns для Категории и Сложности). Кнопка `+ Add to Test Set (Q9)`. Ниже — список недавно добавленных.

### 7. Listener Coach Session (Обратная связь)
- Во время тренировки слушатель видит вопрос от Аватара с указанием его роли и номера вопроса (напр. `АВАТАР (CFO) · Q5/12`).
- Слушатель пишет/говорит свой ответ.
- **Feedback блок:** Появляется *после* ответа слушателя в чате (не как popup, а как сообщение/блок). Включает в себя оценку (например, `Майже - 1 бал (4/5)`) и текст правильного ответа.
- **Score pill:** Постоянно отображается над чатом (напр. `4/5 · 80%`).
