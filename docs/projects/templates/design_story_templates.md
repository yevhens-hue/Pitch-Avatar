1. О проекте (Context & Goals)
Мы внедряем новый стратегический раздел «Шаблоны презентаций» (Presentation Templates) для снижения барьера входа (TTFP) и устранения эффекта «чистого холста» у новых пользователей. Вместо плоских файлов PPTX шаблоны будут использовать нативные JSON-объекты (текст, изображения, ИИ-сценарии), что позволит редактировать и переводить их в один клик.

Для управления этой библиотекой шаблонов мы также создаем полноценный CRUD-инструмент в Панели администратора, позволяющий привязывать шаблоны к приватным мастер-проектам и настраивать их отображение на главной странице пользователей.

Где это находится: 
- Клиентская часть: Главная панель управления (Home Page), левое меню (Sidebar) -> переход на отдельную страницу Templates, а также кнопка внутри Редактора слайдов.
- Административная часть: Панель администратора -> раздел CONTENT -> пункт меню Project Templates.

3. Что нужно нарисовать (Scope of Work)
Необходимо спроектировать новую экосистему шаблонов, состоящую из 5 ключевых смысловых блоков:

Блок 1: Sidebar & Home Page (Точки входа)
Задача: Привлечь пользователя к работе с шаблонами как из глобального меню, так и прямо с главного дашборда платформы.

Элементы:
- Sidebar: Новый пункт меню Templates (с иконкой сетки слоев) в трех состояниях (Default, Hover, Active с выделением рамкой индиго). Размещается между Presentations и AI Chat-avatar.
- Home Page Block: Промо-раздел быстрого старта "Create from Template" с горизонтальной каруселью самых популярных шаблонов и кнопкой View All Templates. Отображение шаблонов в карусели и их порядок настраиваются администратором.
Снимок экрана 2026-05-11 в 14.18.33.png

Блок 2: Templates Gallery (Галерея шаблонов)
Задача: Дать пользователю удобную отдельную страницу со всей базой шаблонов, быстрым поиском и фильтрацией. Для обычного (не-администратора) пользователя кнопка создания шаблона "+ Add Template" в интерфейсе галереи отсутствует.

Элементы:
- Заголовок страницы с бейджем-счетчиком готовых шаблонов (например, "10 ready-to-use").
- Поисковая строка "Search by name or tag..." с иконкой лупы и подсветкой при фокусе.
- Категорийные фильтры-пилюли (Pills) с горизонтальным скроллом и эмодзи (All, HR, Internal Communications, Marketing, Sales, Support, Compliance, IT Security) в активном и неактивном состояниях.
- Сетка интерактивных карточек шаблонов. Карточка содержит:
  * Обложку с 3D-иконкой и бейджем количества слайдов (например, `10 slides` внизу справа).
  * Тег категории (например, `HR` слева над заголовком).
  * Заголовок и краткое описание шаблона.
- **Интерактивное состояние (Hover-эффект):** При наведении на обложку плавно накладывается полупрозрачный индиго-оверлей с двумя кнопками, расположенными строго по вертикали по центру:
  * **Use template** — основная синяя кнопка (вызывает модальное окно создания презентации).
  * **Preview slides** — второстепенная прозрачная кнопка с белым контуром (открывает окно детального просмотра слайдов).
  * Функция дублирования (Duplicate / Копировать) при наведении на карточку шаблона полностью отсутствует.

Блок 3: Editor Toolbar & Apply Flow (Интеграция в Редактор)
Задача: Позволить пользователю бесшовно сменить или применить шаблон прямо во время редактирования презентации.

Элементы:
- Кнопка Templates на верхней панели редактора слайдов (Editor Top Header).
- Выдвижной боковой Drawer (мини-галерея), открывающийся справа при клике на кнопку. Содержит поиск, категории и список доступных шаблонов.
- Диалог подтверждения (Confirmation Dialog) с предупреждением о том, что новый шаблон изменит разметку, но сохранит медиа- и текстовый контент пользователя.

Блок 4: Preview & Creation Dialogs (Окна предпросмотра и создания)
Задача: Продемонстрировать структуру шаблона и позволить пользователю задать имя нового проекта перед переходом в редактор.

Элементы:
- **Модальное окно предпросмотра (Preview Modal):**
  * Верхняя часть с бейджем категории (например, `HR` в левом верхнем углу) и кнопкой закрытия `X`.
  * Центральная большая область обложки шаблона с 3D-иконкой по центру и счетчиком слайдов (например, `10 slides` внизу справа).
  * Горизонтальная карусель миниатюр слайдов под обложкой с номерами и названиями (например, `1 Welcome to the Team!`, `2 Our Mission & Core Values`).
  * Теги под каруселью миниатюр (например, `HR`, `TRAINING`, `POPULAR`).
  * Заголовок шаблона в крупном начертании и детальное текстовое описание.
  * Правый нижний угол: Первичная кнопка **Use this template →** для запуска создания проекта.
- **Окно создания презентации (Create new presentation Dialog):**
  * Открывается при клике на "Use template" (в галерее или превью).
  * Текстовое поле ввода названия презентации: автозаполняется по маске `[Название шаблона] - Project` (например, `Onboarding - Project`).
  * Иконка сверкающих искр по центру.
  * Текст сообщения: *"You are about to create a new interactive presentation based on the Onboarding template. The template contents will be loaded into your editor."*
  * Внизу слева: Две кнопки — основная синяя **Start Editing** и текстовая ссылка **Cancel** рядом с ней.
- **Инструменты создания (для холста):** Селектор форматов (Horizontal 16:9 / Vertical 9:16 / Square 1:1) и панель зумирования холста (Zoom In/Out, Fit to screen).

Блок 5: Admin Panel: Project Templates (CRUD & Настройки)
Задача: Дать администраторам инструмент для наполнения базы данных шаблонов, привязки их к мастер-проектам и управления отображением на главной странице.

Элементы:
- Раздел в боковом меню админки: CONTENT -> Project Templates (выделен синим в активном состоянии).
- Таблица шаблонов Project Templates:
  * Заголовок и описание "Select existing projects to use as templates".
  * Кнопка в правом верхнем углу "+ Add Template" (первичная синяя кнопка).
  * Колонки таблицы: Thumbnail (миниатюра шаблона), Name (название и описание под ним), Source Project (привязанный приватный проект, к которому имеет доступ только админ), Status (бейдж "Active"/"Inactive"), Actions (иконка шестеренки для редактирования/удаления).
  * Пагинация и настройка количества строк (Rows 20) внизу таблицы.
- Модальное окно создания и редактирования (Edit Template):
  * Поля формы:
    - Name (Текстовое поле для названия шаблона).
    - Description (Многострочное текстовое поле для описания).
    - Source Project (Выпадающий список / Dropdown для выбора приватного мастер-проекта, который будет клонироваться пользователям при создании).
    - Home Page Display Settings: Переключатель (Toggle) отображения шаблона на Home Page и числовое поле ввода для настройки порядка сортировки (вес/приоритет отображения).
  * Кнопки Cancel (отмена) и Update/Create (сохранение изменений).
