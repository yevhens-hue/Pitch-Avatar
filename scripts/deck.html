Слайд 7 · Sara в действии

Sara: контекстная помощь в критический момент

Интерфейс на стенде

Аватар проговаривает подсказки вслух — Web Speech API. Сворачивается в FAB.

Почему это важно

  • Самый резкий отвал: Step 1→2 за первые 7 секунд (−35%)
  • Sara включается именно в этот момент
  • Снижает когнитивную нагрузку через быстрые ответы
  • Контекст меняется по шагам — не generic-помощник

UX-особенности реализации

🗣️ TTS Voice
Web Speech API — проговаривает подсказки
🖱️ Drag & Move

📌 Persistent
Чипсы не пропадают — можно задать несколько вопросов
🔇 Click-outside

Слайд 8 · Аналитика и трекинг

PostHog: событийная аналитика настроена

Ключевые события

chat_avatar_rendered — пользователь увидел аватара (Aha! Moment)
chat_avatar_message_sent — первый вопрос к аватару (Activation)
chat_avatar_tour_triggered — запуск интерактивного тура
sara_chip_clicked — клик по быстрому вопросу Sara
wizard_step_completed — завершён шаг в мастере создания

Сквозные свойства (Properties)

  • user_id — строим cohort analysis
  • main_goal — воронки в разрезе бизнес-целей
  • current_url — путь пользователя по шагам
  • is_mobile — сегментация по устройствам

Следующий шаг

  • A/B тест Sara виджета: с/без — измерить delta Step 2 drop-off
  • Funnel: Signup → Step1 → Step2 → Preview (Aha Moment)

Слайд 10 · Стратегия Retention & Growth Loops — Месяц 3

Growth Loops: от инструмента к системе роста

🤝
Social Loops

Коллаборация внутри команд — HR и Sales. Приглашение коллег сразу после создания контента запускает органическую петлю роста.

  • Создал презентацию → Пригласил коллегу
  • HR Learning Epic: командные шаблоны
  • Общий проект → возврат через уведомления
🚀
Virality

«Powered by Pitch Avatar» на каждом расшаренном видео — драйвер новых регистраций без маркетинговых затрат.

  • Виральные петли обмена ссылками
  • Branded footer → viral loop
  • Нулевые CAC для реферальных юзеров
📊
Retention Focus

Анализ когорт — почему не возвращаются после 1-й сессии. Персонализированные триггеры через ценность, не спам-пуши.

  • Когортный анализ в PostHog
  • Контекстные напоминания о незавершённом
  • Возврат через ценность → ↑ W2 Retention
