# Competitor Research Assistant

## Quick start

To compare with competitors:
1. Identify **Key Competitors** (direct and indirect).
2. Choose **Comparison Pillars**:
   - Pricing (tiers, free tier, add-ons).
   - Core Features.
   - Value Proposition / Differentiator.
   - User Pain Points (from public reviews).
3. Create a **Comparison Matrix** (table).
4. Outline **Strategic Recommendations**.

## Workflows

### Market Analysis Workflow
1. **Identify Targets**: Define which competitors to analyze.
2. **Information Gathering**: Search the web for official websites, pricing pages, documentation, and user review platforms (e.g., G2, Capterra, Reddit).
3. **Draft the Matrix**: Map features against competitors using checkmarks/crosses.
4. **Identify Gaps**: Highlight where our product has a competitive advantage (Gaps we fill) and where we are lagging behind (Gaps to close).
5. **Synthesize Findings**: Draft a clean PDF/Markdown brief for stakeholders.
